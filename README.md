Deskripsi Singkat: Program ini digunakan untuk mengelola data antrian pasien dengan cara yang sederhana, dimana user dapat memasukkan nama pasien dan melihat urutan antrian yang sudah tersimpan. Menu ini memungkinkan interaksi langsung antara user dan program untuk melakukan berbagai operasi terhadap data pasien.

Dari segi algoritma dan struktur data, program ini menggunakan list (array) sebagai media penyimpanan data pasien dimana mengunakan ukuran tetap. Pengolahan data dilakukan melalui perulangan (for I in range) untuk mengakses setiap elemen dan percabangan (if-elif) untuk mengatur alur berdasarkan input pengguna.
