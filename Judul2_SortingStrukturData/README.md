Deskripsi Singkat: 
Program ini dipakai untuk memasukkan jumlah siswa dan nilai mereka satu per satu, lalu semua nilai itu disimpan dan ditampilkan dulu sebelum diurutkan. Setelah itu, program akan mengurutkan nilai dari yang paling tinggi ke paling rendah dengan cara membandingkan dan menukar posisi angka, sehingga akhirnya muncul urutan peringkat nilai siswa secara otomatis.
